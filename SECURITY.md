# Security Policy

## Reporting a Vulnerability

If you believe you have found a security issue, please report it privately.

### Urgent Issues

Email `security@mog.capital` and include:
- A clear description of the issue.
- The assessed severity (low / medium / high / critical).
- A minimum reproducible example (steps, code, or request details).

We will acknowledge receipt within 48 hours and work with you on a coordinated fix.
